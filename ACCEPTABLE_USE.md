# Acceptable Use Policy

VulnoraIQ is provided for defensive, authorised AI security assessment only.

## Permitted use

You may use VulnoraIQ for:

- testing systems you own;
- testing systems you are explicitly authorised to assess;
- safe local demo testing;
- internal security validation;
- defensive control testing;
- evidence collection for internal review.

## Prohibited use

You must not use VulnoraIQ for unlawful activity, unauthorised testing, bypassing access controls, violating third-party terms, or any use outside the scope of written permission from the system owner.
